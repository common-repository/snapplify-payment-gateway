Snapplify Payment infrastructure built specifically for the education sector, ensuring easy and secure payment options for schools and parents.

== Description ==
<strong>Get paid anywhere and in local currencies</strong><br><br>
With Snapplify Pay, you can accept payments across multiple regions and in local currencies. Local payments across 46 countries together with cross-border payments means that you have the confidence of getting paid anywhere.<br><br>
<strong>Integrated payments for education</strong><br><br>
Easy payment options for schools and parents are essential, that’s why our payment infrastructure is built specifically for the education sector. Through its integration with major payment gateways, Snapplify Pay supports debit and credit card payments, EFT and Instant EFT payments, mobile-phone-based financial services like M-Pesa and payments using Snapplify credit.<br><br>
<strong>Convenient and safe</strong><br><br>
Efficient and secure payment facilities for your customers.<br><br>
<strong>Reach new markets</strong><br><br>
Snapplify Pay enables cross-border payments. Customers across 46 countries can pay you in their local currency, allowing you to grow your business in new and existing markets.<br><br>
<strong>Save hours on development</strong><br><br>
Use Snapplify Pay immediately, without having to develop your own infrastructure from scratch

== Installation ==
= Minimum Requirements =

WooCommerce is required in order to use this plugin, in line with that;
* PHP 7.2 or greater is recommended

Additional setup is required in conjunction with Snapplify in order to register your plugin with us. Please feel free to reach out to us on help@snapplify.com.
Visit the [WooCommerce server requirements documentation](https://docs.woocommerce.com/document/server-requirements/) for a detailed list of server requirements.

= Automatic installation =

Automatic installation is the easiest option -- WordPress will handle the file transfer, and you won’t need to leave your web browser. To do an automatic install of Snapplify Payment Gateway, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”

In the search field type “Snapplify Payment Gateway,” then click “Search Plugins.” Once you’ve found us,  you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by! Click “Install Now,” and WordPress will take it from there.

= Manual installation =

The manual installation method requires downloading the Snapplify Payment Gateway plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).
